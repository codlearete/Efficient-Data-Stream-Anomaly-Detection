Libraries Used:
**Dash**: A web application framework for building interactive web applications.
**html, dcc from Dash**: Modules for creating HTML components and interactive components.
**plotly.graph_objs as go**: Plotly library for creating interactive and dynamic plots.
**numpy (np)**: Library for numerical operations and generating synthetic data.
**collections.deque**: Double-ended queue for efficiently managing a fixed-size sequence of data.
